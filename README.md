# Workify

Workify is a platform to add monitor and analyze your **Daily Work Hours** . 

## Features
- [x] Clock-In & Clock-Out of work
- [ ] Display work hours on day to day basis
- [ ] Sort data according to date, work type etc.
- [ ] Edit previous data
- [ ] Visualize work hours
